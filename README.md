# Microposts App

A JavaScript Frontend CRUD Posts App using ES6 Modules & JSON Server for the REST API. Code is transpiled using Webpack & Babel.

### Version
1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```

### Build
Compile and build

```sh
$ npm run build
```
